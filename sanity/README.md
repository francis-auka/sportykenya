# Sanity Studio

This is the Sanity Studio for the SportyKenya project.

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Configure Project ID**:
    Open `sanity.config.ts` and `sanity.cli.ts` and replace `'YOUR_PROJECT_ID'` with your actual Sanity Project ID.

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## Project Structure

-   `sanity.config.ts`: Main configuration for the Studio.
-   `sanity.cli.ts`: Configuration for the Sanity CLI.
-   `schemaTypes/`: Directory for your content schemas.
